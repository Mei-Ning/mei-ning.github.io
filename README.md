## 自我介紹
### 黃美寧 / Melissa

- 擁有多項能力，從介面設計、前端到後端資料庫。
- 服裝設計畢業，卻對網頁設計充滿熱情，透過職訓局培訓，開始接觸程式設計。
- 有自主學習與團隊合作專案經驗。

大學畢業後，在服裝產業有近五年的經驗，服裝是我最熟悉不過的事情，因為太熟悉反而知道服裝產業的落寞。在半年前父親中風需要照顧，我毅然決然北上照顧行動不便的父親，在這段時間，我偶然的看到youtuber在介紹UI設計，影片中網頁設計的作品，美到令人雞皮疙瘩，開始對這個行業產生幻想，便開始蒐集資訊，過程中更確立這是自己喜歡的路，並開始在職訓局的全端工程師培訓。

課程上從介面設計、前端工程、後端資料庫。尤其著重UI設計、前端網頁技術及後端資料庫CRUD，並在團體專案磨練UI Design及前端須具備的各項技能，成果發表得到組員及老師讚賞我的設計，感到非常有成就感。

雖然非資工本科，但我決定踏出服裝的舒適圈，往網頁設計發展，相信未來職涯有更多挑戰，需要透過不斷學習與經驗累積。而我的理想是希望未來在掌握前端技術，更能實現UI，並結合我以前服裝設計的技巧，能夠從中發展更多的可能性，做出厲害的作品，進而在這時代展現自己的價值。
